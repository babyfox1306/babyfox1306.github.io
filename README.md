# GitHub Pages Root Domain

Repo này để host file `app-ads.txt` ở root domain cho AdMob verification.

## Setup

1. Tạo repo mới trên GitHub với tên chính xác: `babyfox1306.github.io`
2. Push các file trong folder này lên repo đó
3. Bật GitHub Pages: Settings → Pages → Source: `main` branch
4. File `app-ads.txt` sẽ có tại: `https://babyfox1306.github.io/app-ads.txt`

## Files

- `app-ads.txt` - AdMob verification file (phải ở root domain)
